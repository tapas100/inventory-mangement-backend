# Inventory Management API with Adonis

![](adonis.png)


## Getting Started
  
  Install Node modules 
  
  ```js
  npm install
```
  
  Start MongoDB  
  
  ```js
sudo service mongod start
```
  
  Run the development server 
  
  ```js
 adonis serve --dev
```
